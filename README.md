# Group-5-Pemrograman-A
## Compile
`gcc predictor2.c -o predictor2.exe -lm`
## How to Use
### With a CSV file:
`predictor2.exe data.csv`

### With standard input (pipe or redirect):
`type data.csv | predictor2.exe` \
or \
`predictor2.exe < data.csv`

### Change the filename to your needs.
